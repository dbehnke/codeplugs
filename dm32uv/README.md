# Baofeng DM-32UV Codeplug Resources

This directory contains resources for programming the Baofeng DM-32UV radio, specifically for the Mi5 DMR network.

## Files

* **[roam-channels.csv](roam-channels.csv):** A list of Mi5 Network repeater channels formatted for import or manual entry. Includes Frequency, Color Code, and Time Slot information.
* **[roam-zones.csv](roam-zones.csv):** Defines Roaming Zones for the Mi5 Network, grouping channels by geographic area (e.g., Detroit Area, Flint/Saginaw).
* **[roaming_instructions.md](roaming_instructions.md):** Step-by-step guide on how to set up roaming on the DM-32UV using these resources.

## External Resources

* **[M7OCM/DM-32UV GitHub Repository](https://github.com/M7OCM/DM-32UV):** A valuable resource for DM-32UV tools, firmware, and information.
